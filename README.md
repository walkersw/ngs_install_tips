# ngs_install_tips

          Compiling and Installing NGSolve Under WSL.

                     (c) Shawn W. Walker

DESCRIPTION
========================================================================

This is a collection of tips and instructions for compiling and installing NGSolve from source under Windows Subsystem Linux (WSL).

USAGE
========================================================================

1. NGSolve can be found here: http://ngsolve.org/

2. These instructions are based on using the WSL system under Windows.

3. The Wiki is here:  https://github.com/walkersw/ngs_install_tips/wiki

COMPATIBILITY NOTES
========================================================================
This should work with Windows 10 or 11.
